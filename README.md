# Japanese Language Pack for Flarum (beta8)

Japanese Language pack extension for [Flarum](http://flarum.org).(v0.1.0-beta8)

## Installation

```shell
composer require n04h/flarum-ext-japanese
```

## Uninstall

```shell
composer remove n04h/flarum-ext-japanese
```
